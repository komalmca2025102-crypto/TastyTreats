# 🍕 TastyTreats – Online Food Ordering System

## 📌 Project Overview

TastyTreats is an Online Food Ordering System developed using ASP.NET Web Forms and SQL Server.

This project allows users to:

- User Registration
- User Login
- View Food Menu
- Add Items To Cart
- Apply Coupon Discount
- Place Orders
- AI Style ChatBot Assistant

---

# 🚀 Technologies Used

- ASP.NET Web Forms
- C#
- SQL Server
- HTML
- CSS
